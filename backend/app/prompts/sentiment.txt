# 利好/利空/中性判断 Prompt (v1.2)
# 输出新闻对A股市场的影响方向

## 核心原则

**一般利好 ≠ A股利好**

你必须判断的是：这条新闻是否会造成A股相关资产的价格变动。

判断标准是新闻是否能通过某种路径影响A股投资者的买入/卖出决策。

---

- 影响对象: {affected_objects}

- 政策方向: {policy_direction}

- 溯源判断结果: {source_verification_result}

## 第一部分：个股强利好消息类型库（sentiment_score ≥ 60）

### 1. 重大资产重组（S级利好）
**典型表述**：
```
筹划重大资产重组、发行股份购买资产、资产置换、注入优质资产、
重组方案获批、重大资产重组完成、并购重组恢复审核、
重组标的属于热门赛道（AI/算力/半导体/机器人/低空经济/商业航天）
```
**高强度情形**：
- 传统业务公司重组进入热门方向
- 亏损公司注入盈利资产
- 小市值公司收购大体量优质资产
- 国资或产业资本主导重组
- 重组停牌较久，市场无法提前交易

**常见市场反应**：复牌一字板、连续涨停、大幅高开

---

### 2. 并购收购
**典型表述**：
```
拟收购某公司股权、收购某公司控股权、现金收购资产、
发行股份收购资产、收购热门赛道公司、收购完成、取得目标公司控制权
```
**高强度情形**：
- 收购标的盈利能力强
- 收购标的属于热门题材
- 交易金额较大
- 收购后形成控股
- 收购标的与上市公司形成产业协同

**降权情形**：只是意向协议、交易金额很小、标的亏损严重

---

### 3. 控制权变更（S级利好）
**典型表述**：
```
控股股东变更、实际控制人变更、国资入主、央企入主、
产业资本入主、战略投资者入股、表决权委托、协议转让股份
```
**高强度情形**：
- 地方国资/央企/知名产业资本入主
- 新实控人有资产注入预期
- 原公司基本面较差，新股东资源强

**常见市场反应**：一字板、连续涨停

---

### 4. 大额订单/重大合同（业绩兑现型利好）
**典型表述**：
```
签订重大合同、签订特别重大合同、中标重大项目、收到中标通知书、
获得大额订单、长期供货协议、进入核心客户供应链、成为独家供应商
```
**判断强度关键**：
```
合同金额/公司上一年度营业收入：
- 超10%：明显利好
- 超30%：强利好
- 超50%：极强利好

客户质量：央企/国企/头部企业/海外巨头 加分
订单来自热门方向（AI/军工/算力/机器人/商业航天）加分
```

**弱化情形**：框架协议、意向协议、无具体金额、履约周期很长

---

### 5. 业绩暴增（S级利好）
**典型表述**：
```
业绩预增、净利润同比大幅增长、净利润增长300%以上、
扣非净利润大幅增长、创历史新高、业绩大超预期、盈利能力显著改善
```
**判断标准**：
```
净利润同比增长：
- 50%以上：中等利好
- 100%以上：强利好
- 300%以上：极强利好
- 500%以上：可能触发强烈高开或涨停
```

**更强情形**：主营业务带动增长、扣非净利润同步大增、此前市场预期较低

**降权情形**：增长来自出售资产/政府补助/投资收益/会计处理（不可持续）

---

### 6. 业绩扭亏为盈
**典型表述**：
```
预计扭亏为盈、实现扭亏、扣非后扭亏、
经营情况明显改善、主营业务盈利能力恢复
```
**高强度情形**：
- 连续亏损后首次扭亏
- 扣非净利润也扭亏
- 主营业务改善导致扭亏
- 扭亏后可能摘帽

---

### 7. 摘帽/撤销风险警示
**典型表述**：
```
申请撤销退市风险警示、撤销退市风险警示、撤销其他风险警示、
股票简称变更、股票恢复正常涨跌幅限制、摘帽
```
**高强度情形**：摘帽+业绩改善、摘帽+重组预期、摘帽+国资入主、摘帽+热门题材

---

### 8. 产品获批/认证通过/资质突破
**典型表述**：
```
产品获批上市、获得注册证、通过认证、取得生产许可证、
通过客户验证、进入供应商名单、获得适航认证、
取得军工资质、通过临床试验、临床III期达到主要终点
```
**高强度情形**：核心产品获批、产品市场空间大、此前存在较大不确定性、有望快速放量

---

### 9. 新产品发布/技术突破/量产落地
**典型表述**：
```
技术取得重大突破、产品实现量产、全球首款、国内首创、
首次商业化应用、核心技术突破、良率大幅提升、成本大幅下降、正式投产
```
**热门方向**：AI芯片、光模块、机器人、固态电池、低空经济、商业航天、创新药、半导体设备、光刻胶、军工装备

---

### 10. 回购股份
**典型表述**：
```
拟回购股份、大额回购、注销式回购、回购用于注销、
回购金额上调、回购价格上限较高、已完成回购
```
**强度排序**：注销式回购 > 大额回购 > 提高回购金额 > 普通回购 > 象征性回购

---

### 11. 大股东/实控人/高管增持
**典型表述**：
```
控股股东增持、实际控制人增持、董事长增持、高管集体增持、
增持计划、承诺不减持、延长限售期
```
**强度排序**：实控人大额增持 > 控股股东增持 > 董事长增持 > 高管集体增持

---

### 12. 进入核心客户供应链
**典型表述**：
```
进入某头部客户供应链、获得客户定点、取得项目定点通知书、
成为合格供应商、通过某客户认证、向某龙头企业供货
```
**高强度客户**：苹果、华为、英伟达、特斯拉、比亚迪、宁德时代等龙头

## 第二部分：个股强利空消息类型库（sentiment_score ≤ -60）

### 1. 被ST/*ST/实施风险警示（S级利空）
**典型表述**：
```
实施退市风险警示、实施其他风险警示、股票简称变更为*ST、
股票简称变更为ST、被实施风险警示、复牌后实施退市风险警示
```
**常见市场反应**：一字跌停、连续跌停、大幅低开

---

### 2. 可能退市/终止上市（S级利空）
**典型表述**：
```
可能被终止上市、收到终止上市事先告知书、股票进入退市整理期、
触及交易类退市指标、触及财务类退市指标、触及重大违法退市
```
**常见市场反应**：大幅低开、一字跌停、连续跌停

---

### 3. 业绩暴雷/巨亏
**典型表述**：
```
业绩预亏、净利润亏损、由盈转亏、亏损扩大、
净利润同比大幅下降、扣非净利润亏损、业绩低于预期
```
**判断强度**：
```
- 净利润下降超50%：明显利空
- 由盈转亏：强利空
- 巨额亏损：极强利空
- 亏损触发ST风险：最高风险
```

---

### 4. 商誉减值/资产减值
**典型表述**：
```
计提商誉减值、计提资产减值准备、存货跌价准备、
信用减值损失、长期股权投资减值、固定资产减值
```
**高强度情形**：减值金额巨大、导致公司由盈转亏、导致净资产下降、可能触发ST

---

### 5. 审计非标/无法表示意见（S级利空）
**典型表述**：
```
被出具保留意见、被出具否定意见、被出具无法表示意见、
内控审计否定意见、审计报告非标准意见、持续经营能力存在重大不确定性
```
**强度排序**：无法表示意见 > 否定意见 > 保留意见 > 带强调事项段

---

### 6. 被证监会立案调查（S级利空）
**典型表述**：
```
收到中国证监会立案告知书、因涉嫌信息披露违法违规被立案、
涉嫌财务造假、涉嫌内幕交易、涉嫌操纵市场
```
**高强度情形**：涉及财务造假、涉及重大信披违法、涉及实控人、涉及多年财报、涉及重大违法退市风险

---

### 7. 行政处罚/监管处罚
**典型表述**：
```
收到行政处罚事先告知书、收到行政处罚决定书、
受到交易所纪律处分、收到监管函、收到警示函
```
**强度排序**：行政处罚决定书 > 行政处罚事先告知书 > 纪律处分 > 监管函 > 问询函

---

### 8. 并购重组失败（S级利空）
**典型表述**：
```
终止重大资产重组、终止发行股份购买资产、终止收购、
重组失败、交易双方未能达成一致、重组审核未通过、撤回申请文件
```
**高强度情形**：重组标的属于热门资产、股价此前已有大涨、公司本身基本面较差

---

### 9. 大股东减持/清仓式减持
**典型表述**：
```
股东拟减持、控股股东拟减持、实控人拟减持、
董监高拟减持、清仓式减持、减持不超过公司总股本若干比例
```
**强度排序**：实控人减持 > 控股股东减持 > 董事长减持 > 高管减持
清仓式减持 > 大比例减持 > 小比例减持

---

### 10. 股权质押爆仓/司法冻结
**典型表述**：
```
控股股东股份被司法冻结、股份被轮候冻结、股份将被司法拍卖、
质押股份触及平仓线、被动减持、强制平仓、债务违约
```
**高强度情形**：冻结股份比例高、涉及控股股东或实控人、可能导致控制权变更

---

### 11. 实控人/董事长/高管出事
**典型表述**：
```
实际控制人被留置、董事长被调查、高管失联、
实控人被采取强制措施、涉嫌违法犯罪、不能正常履职
```
**高强度情形**：实控人深度参与经营、公司依赖实控人资源、涉及资金占用或财务造假

---

### 12. 债务违约/流动性危机
**典型表述**：
```
债务逾期、未能按期偿还债务、票据逾期、银行账户被冻结、
流动性紧张、债券违约、无法兑付、进入预重整、破产重整
```
**高强度情形**：债务金额巨大、银行账户被冻结、主营业务停滞、可能触发退市风险

---

### 13. 安全事故/停产整改
**典型表述**：
```
发生安全事故、重大事故、爆炸、火灾、污染事件、
停产整改、接受调查、生产基地停产、产品召回
```
**高强度情形**：造成人员伤亡、核心工厂停产、监管部门介入、产品大规模召回

## 判断规则

1. **无法找到A股影响对象 → sentiment_score ≤ 30**
2. **S级利好 → sentiment_score ≥ 85**
3. **S级利空 → sentiment_score ≤ -85**
4. **间接传导必须追溯到底**：不能跳过传导链条
5. **利好利空共存时**：分别输出，由 multi_mixed 标记
6. **传导路径不明确时**：保守给中性分
7. **意向/框架协议**：降权一级
8. **时间敏感**：已过时效的新闻不给高分
9. **超预期程度**：增长越多/金额越大/级别越高 → 分数越高
