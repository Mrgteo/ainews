# 舆情监控与分析汇总系统 - 开发方案

## 一、项目概述

### 1.1 项目目标
构建一个智能舆情监控与分析系统，实现对同花顺财经电报新闻的自动化抓取、AI智能分析与分类、可视化展示的一体化解决方案。

### 1.2 核心功能
- 新闻自动化抓取（Selenium + ChromeDriver）
- AI驱动的新闻重要性评分
- 增量信息识别与预期差分析
- 多维度综合评分与智能分类
- 可视化交互界面

#### 3.2.2 Prompt设计

**重要性评分Prompt:**
```
你是一个专业的金融新闻分析师。请分析以下新闻的重要性：

新闻标题：{title}
新闻内容：{content}

请从以下维度进行评估：
1. 重要性 (0-100): 该新闻对市场的影响程度
2. 增量性 (是/否): 是否包含新的、未被市场充分消化的信息
3. 预期差: 超预期/符合预期/低于预期/无法判断
4. 分类: 行业/公司/宏观/市场/其他

请以JSON格式输出分析结果。
```

#### 3.2.3 评分标准

| 评分维度 | 说明 | 权重 |
|----------|------|------|
| 基础重要性 | 新闻本身的新闻价值 | 30% |
| 增量属性 | 是否是新信息 | 25% |
| 预期差 | 实际vs预期 | 25% |
| 市场反应 | 潜在影响范围 | 20% |

---

### 3.3 综合评分算法

#### 3.3.1 评分公式
```
综合分数 = 重要性分数 × 0.4
         + 增量系数 × 30  // 增量=1.0, 部分增量=0.5, 非增量=0.2
         + 预期差系数 × 20 // 超预期=20, 符合=10, 低于=0
         + 市场敏感度 × 10 // 根据新闻类型浮动
```

#### 3.3.2 新闻分类

| 类别 | 综合分数范围 | 标签颜色 | 说明 |
|------|-------------|----------|------|
| 紧急 | 80-100 | 红色 | 需立即处理 |
| 重点关注 | 60-79 | 橙色 | 需要重点关注 |
| 一般参考 | 40-59 | 蓝色 | 常规参考信息 |
| 已处理/噪音 | 0-39 | 灰色 | 低价值信息 |

### 9.1 爬虫核心代码结构
```python
class TonghuashunScraper:
    def __init__(self):
        self.base_url = "https://news.10jqka.com.cn/telegraph/"
        self.options = ChromeOptions()
        self.options.add_argument('--headless')
        self.options.add_argument('--no-sandbox')

    def scrape(self, days=3):
        """抓取近N天新闻"""
        driver = webdriver.Chrome(options=self.options)
        news_list = []

        try:
            driver.get(self.base_url)
            # 等待页面加载
            WebDriverWait(driver, 10).until(
                EC.presence_of_element_located((By.CSS_SELECTOR, ".news-list"))
            )

            # 滚动加载更多
            self._load_all_news(driver)

            # 解析新闻列表
            news_elements = driver.find_elements(By.CSS_SELECTOR, ".news-item")
            for element in news_elements:
                news = self._parse_news_item(element)
                if self._is_within_days(news['pub_time'], days):
                    news_list.append(news)

            # 获取完整内容
            for news in news_list:
                news['content'] = self._get_full_content(news['url'])

        finally:
            driver.quit()

        return news_list
```

### 9.2 AI分析核心代码
```python
class NewsAnalyzer:
    def __init__(self, api_key: str):
        self.client = OpenAI(api_key=api_key, base_url="https://api.deepseek.com")

    async def analyze_news(self, title: str, content: str) -> dict:
        """AI分析新闻"""
        prompt = f"""
        你是一个专业的金融新闻分析师。请分析以下新闻：

        新闻标题：{title}
        新闻内容：{content}

        请评估：
        1. 重要性分数 (0-100)
        2. 是否是增量信息 (是/否)
        3. 预期差 (超预期/符合预期/低于预期/无法判断)
        4. 核心要点 (列出2-3个)
        5. 分类 (行业/公司/宏观/市场/其他)

        请以JSON格式输出。
        """

        response = self.client.chat.completions.create(
            model="deepseek-chat",
            messages=[{"role": "user", "content": prompt}]
        )

        return self._parse_response(response)
```

### 9.3 综合评分算法
```python
def calculate_final_score(analysis_result: dict) -> float:
    """计算综合评分"""
    importance = analysis_result.get('importance_score', 50)

    # 增量系数
    is_incremental = analysis_result.get('is_incremental', False)
    incremental_factor = 1.0 if is_incremental else 0.3

    # 预期差系数
    expectation_map = {
        '超预期': 20,
        '符合预期': 10,
        '低于预期': 0,
        '无法判断': 5
    }
    expectation_diff = analysis_result.get('expectation_diff', '无法判断')
    expectation_factor = expectation_map.get(expectation_diff, 5)

    # 市场敏感度 (根据分类浮动)
    category_sensitivity = {
        '宏观': 10,
        '行业': 8,
        '公司': 6,
        '市场': 7,
        '其他': 3
    }
    category = analysis_result.get('category', '其他')
    sensitivity = category_sensitivity.get(category, 3)

    # 综合评分
    final_score = (
        importance * 0.4 +
        incremental_factor * 30 +
        expectation_factor +
        sensitivity
    )

    return round(min(100, max(0, final_score)), 2)


def determine_news_type(final_score: float) -> str:
    """根据分数确定新闻类型"""
    if final_score >= 80:
        return "紧急"
    elif final_score >= 60:
        return "重点关注"
    elif final_score >= 40:
        return "一般参考"
    else:
        return "已处理噪音"
```
